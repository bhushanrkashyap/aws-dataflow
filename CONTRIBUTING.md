# Contributing

Thank you for considering contributing to this project. Contributions should follow these guidelines:

- Fork the repository and create a feature branch for your change.
- Ensure code is formatted (use `black` for Python), and run unit tests locally.
- Open a Pull Request with a clear title and description explaining the change and motivation.
- Add tests for any bug fixes or new features.

Maintainers will review PRs and may request changes before merging.
